## 分布式锁实现的技术

基于数据实现分布式锁
 性能较差，容易出现单点故障
 锁没有失效事件,容易死锁。
 非阻塞式
 不可重入
 
基于缓存实现分布式锁
 锁没有失效事件,容易死锁
 非阻塞式
 不可重入
 
基于Zookeeper实现分布式锁
 实现相对简单
 可靠性高
 性能较好
